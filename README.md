# Amit Panda — Portfolio

## Quick Start

1. Install dependencies:
   npm install

2. Run locally:
   npm start

3. Build for production:
   npm run build

## Deploy to Netlify
- Run `npm run build`
- Drag the `build/` folder to netlify.com

## Deploy to Vercel
- Push this folder to GitHub
- Connect repo on vercel.com — done!

## Customization
- All your data is in the `DATA` object at the top of `src/App.jsx`
- Update GitHub links, add photos, embed YouTube videos in the project detail sections
